## Project Description
his project is an audit of open-source software (Python). It includes analysis and 5 shell scripts demonstrating Linux concepts.

## Scripts Overview

### Script 1: System Identity Report
Displays system information like kernel, user, uptime, date, and Linux distribution.

### Script 2: FOSS Package Inspector
Checks if Python is installed and shows package details.

### Script 3: Disk and Permission Auditor
Displays directory permissions, ownership, and disk usage.

### Script 4: Log File Analyzer
Analyzes log files and counts occurrences of a keyword.

### Script 5: Open Source Manifesto Generator
Generates a personalized open-source manifesto.

## How to Run Scripts

### Step 1: Make scripts executable
chmod +x script1.sh
chmod +x script2.sh
chmod +x script3.sh
chmod +x script4.sh
chmod +x script5.sh

### Step 2: Run scripts
./script1.sh
./script2.sh
./script3.sh
./script4.sh /var/log/dpkg.log error
./script5.sh
